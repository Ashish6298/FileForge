## üå† Getting Started

### Prerequisites
- **Node.js** (v16 or higher)  
- **npm** (v8 or higher)  
- **FFmpeg** (installed globally or via `@ffmpeg-installer/ffmpeg`)  
- A **Gemini API Key** (for summarization features)  

### Installation
1. **Clone the Repository**  
   ```bash
   git clone https://github.com/Ashish6298/FileForge
   cd FileForge
   ```

2. **Install Dependencies**  
   ```bash
   npm install
   ```

3. **Set Up Environment Variables**  
   Create a `.env` file in the root directory and add:  
   ```
   PORT=5000
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the Server**  
   ```bash
   npm start
   ```
   Your server will launch at `http://localhost:5000`. Prepare for liftoff! üöÄ
